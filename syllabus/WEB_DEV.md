# Web Development Syllabus

Complete roadmap for modern full-stack web development covering frontend, backend, databases, and real-time applications.

## üìö Course Modules

### 1. HTML/CSS

- Semantic HTML5
- CSS3 fundamentals and advanced techniques
- Responsive design principles
- Flexbox and Grid layouts
- CSS animations and transitions

### 2. JavaScript Basics

- Variables, data types, and operators
- Functions and scope
- ES6+ features (arrow functions, destructuring, spread/rest)
- Array methods and object manipulation
- DOM manipulation

### 3. JavaScript Architecture

- Event loop and call stack
- Closures and higher-order functions
- Prototypes and inheritance
- Module patterns
- Design patterns in JavaScript

### 4. Async JavaScript

- Callbacks and callback hell
- Promises and promise chaining
- Async/await syntax
- Error handling in asynchronous code
- Working with APIs

### 5. Node vs Browser JavaScript

- Understanding runtime environments
- Browser APIs vs Node.js APIs
- CommonJS vs ES Modules
- window object vs global object
- Event handling differences

### 6. HTTP and Express

- HTTP protocol fundamentals
- Request/response cycle
- Express.js framework setup
- Routing and middleware
- RESTful API design
- Error handling and validation

### 7. Databases and MongoDB

- Database fundamentals (SQL vs NoSQL)
- MongoDB architecture
- CRUD operations
- Schema design and data modeling
- Mongoose ODM
- Indexing and performance optimization

### 8. Postgres + Prisma/Drizzle

- PostgreSQL setup and basics
- SQL queries and joins
- Prisma ORM configuration
- Schema migrations
- Relations and foreign keys
- Query optimization with Prisma
- Alternative: Drizzle ORM

### 9. TypeScript

- Type annotations and inference
- Interfaces and type aliases
- Generics
- Advanced types (union, intersection, conditional)
- TypeScript with React and Node.js
- Configuration and best practices

### 10. Turborepo

- Monorepo architecture
- Setting up Turborepo
- Sharing code between packages
- Build optimization
- Caching strategies
- Managing dependencies

### 11. BunJS

- Introduction to Bun runtime
- Performance benefits over Node.js
- Package management with Bun
- Building applications with Bun
- Migration from Node.js

### 12. React

- Components and props
- State and lifecycle
- Hooks (useState, useEffect, useContext, custom hooks)
- React Router for navigation
- Context API for state management
- Performance optimization (memo, useMemo, useCallback)

### 13. Tailwind CSS

- Utility-first CSS approach
- Tailwind configuration
- Responsive design with Tailwind
- Custom themes and plugins
- Component composition
- Production optimization

### 14. Next.js

- Server-side rendering (SSR)
- Static site generation (SSG)
- API routes
- File-based routing
- Image optimization
- App Router vs Pages Router
- Deployment and performance

### 15. WebSockets + WebRTC

- Real-time communication fundamentals
- Socket.io implementation
- WebSocket protocols
- WebRTC for peer-to-peer connections
- Video/audio streaming
- Chat applications

### 16. Queues/Pubsubs

- Message queue architecture
- Redis for pub/sub
- RabbitMQ basics
- Background job processing
- Event-driven architecture
- Scaling with queues
